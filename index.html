BACnet protocol engineering

An embedded BACnet stack across the product line

I ported and integrated the BACnet communication stack — based on the open-source BACnet Stack project — used on the B5, QIRF-II, and QVRF platforms, covering the MS/TP data link, object model, standard services, alarming, and device configuration. The QVRF implementation was independently verified and achieved BTL Listing at BACnet Protocol Revision 26 (PR26). The BAC-Box gateway extends the same protocol work to native BACnet/IP for the M-Controller and Q-Controller.
